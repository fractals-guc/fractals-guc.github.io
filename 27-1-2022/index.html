Problem of The week

Find all integer solutions to the equation

1a+1b=17 \dfrac{1}{a} + \dfrac{1}{b} = \dfrac{1}{7} a1​+b1​=71​

Solution

Clearing the fractions and rearranging

0=ab−7(a+b) 0 = ab - 7(a+b) 0=ab−7(a+b)

Adding 727^272 and factrorizing, we get

72=ab−7a−7b+72= 7^2 = ab - 7a - 7b + 7^2 = 72=ab−7a−7b+72= (a−7)(b−7)=72 (a-7)(b-7) = 7^2 (a−7)(b−7)=72

now we get all possible factroriziations

72=(±72)(±1)=(±1)(±72)=(7)(7) 7^2 = (\pm 7^2)(\pm 1) = (\pm 1)(\pm 7^2) = (7)(7) 72=(±72)(±1)=(±1)(±72)=(7)(7)

excluding the zero values of a,ba, ba,b Thus we get the five pairs

(7±72,7±1),(7±1,7±72),(14,14) (7\pm 7^2, 7\pm 1), (7 \pm 1, 7 \pm 7^2), (14, 14) (7±72,7±1),(7±1,7±72),(14,14)
